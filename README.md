## 介绍

> [RuoYi-Vue](https://gitee.com/y_project/RuoYi-Vue)的Golang实现，Java选手的Golang奇妙之旅~

基于Gin+Gorm+Vue前后端分离的Go快速开发框架

### 架构

**后端**

- Golang v1.19
- Gin v1.8.1
- Gorm v1.24.0

**前端**

- Vite

- Vue3

- Element-Plus

### 特性

- ...
